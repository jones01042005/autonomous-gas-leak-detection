🔥 Gas Leak Detection System (Edge AI)

📌 Overview
An intelligent gas detection system that uses Edge AI and IoT to detect hazardous gases in real time and trigger immediate safety actions without cloud dependency.

🚀 Features
⚡ Real-time gas detection
🧠 Edge AI-based classification (Safe / Warning / Danger)
🔔 Instant alerts (buzzer + LEDs)
🔌 Automatic safety control (relay)
📊 Data logging with backend server

🏗️ Tech Stack
ESP32 (Microcontroller)
MQ2 & MQ7 Gas Sensors
Arduino IDE (C/C++)
Flask (Python backend)
SQLite / CSV

⚙️ How It Works
Sensors detect gas levels
ESP32 processes data using ML model
Classifies environment (Safe / Warning / Danger)
Triggers alerts and relay actions
Sends data to backend for logging

🌍 Applications
Smart homes
Industrial safety
IoT monitoring systems

🔮 Future Improvements
Mobile alerts (SMS/App)
Advanced ML models
Cloud dashboard integration

⭐ Key Highlight
A real-time Edge AI safety system with fast response, high reliability, and no cloud dependency.
